My primary contribution was the design and implementation of the Minimax AI agent.

The agent uses:

* Depth-limited Minimax search
* Alpha-Beta pruning
* A custom heuristic evaluation function
* Board-state analysis based on offensive opportunities, defensive threats and central-board control

I also participated in testing, integration and benchmarking of the complete system.

The Monte Carlo Tree Search (MCTS) agent was implemented by my teammate.
